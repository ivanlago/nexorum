Demonstração: remover a marcação do fundo do vídeo

Esta versão não usa apenas blend mode. Ela processa o vídeo em canvas e transforma pixels de fundo em transparência, preservando os objetos centrais. Ajuste os controles até o fundo desaparecer sem destruir o objeto.

Orientacao para aplicar no site real

  1. Aplicar somente nos quatro videos da secao [01]: /card1-video.webm, /card2-video.webm, /card3-video.webm e /card4-video.webm.
  2. Manter o video original escondido e desenhar cada frame em um canvas.
  3. Preservar a proporcao vertical dos arquivos: eles sao 720x1280. Nesta demo, o canvas esta em 172x306.
  4. Usar luma key no canvas para transformar o fundo gravado do video em transparencia.
  5. Aplicar mix-blend-mode: lighten no canvas, nao apenas no wrapper.
  6. Remover ou reduzir o halo/blur atras do video, porque ele volta a marcar o fundo.
/* CSS principal */
.cardVideoStage {
  overflow: hidden;
  isolation: isolate;
}

.cardVideoCanvas {
  width: 172px;
  height: 306px;
  mix-blend-mode: lighten;
}

/* Valores usados nesta demo */
const threshold = 92;
const softness = 46;
const boost = 1.28;

/* Desenhar sem deformar */
const videoRatio = video.videoWidth / video.videoHeight;
const canvasRatio = canvas.width / canvas.height;
let drawWidth = canvas.width;
let drawHeight = canvas.height;
let drawX = 0;
let drawY = 0;

if (videoRatio > canvasRatio) {
  drawHeight = canvas.height;
  drawWidth = drawHeight * videoRatio;
  drawX = (canvas.width - drawWidth) / 2;
} else {
  drawWidth = canvas.width;
  drawHeight = drawWidth / videoRatio;
  drawY = (canvas.height - drawHeight) / 2;
}

ctx.drawImage(video, drawX, drawY, drawWidth, drawHeight);

/* Remover fundo por luminancia */
const frame = ctx.getImageData(0, 0, canvas.width, canvas.height);
const data = frame.data;

for (let i = 0; i < data.length; i += 4) {
  const r = data[i];
  const g = data[i + 1];
  const b = data[i + 2];
  const luma = 0.299 * r + 0.587 * g + 0.114 * b;
  const alpha = Math.min(1, Math.max(0, (luma - threshold) / softness));

  data[i] = Math.min(255, r * boost);
  data[i + 1] = Math.min(255, g * boost);
  data[i + 2] = Math.min(255, b * boost);
  data[i + 3] = Math.round(255 * alpha);
}

ctx.putImageData(frame, 0, 0);
